AnyChatMultiCamera	AnyChat C#多路音视频输入示例程序，VS2010工程，演示AnyChat的多路音视频输入功能，展示了可以对系统内连接的多个摄像头的视频数据进行显示的应用场景；

HelloAnyChatCloud	AnyChat C# 多种登录方式的示例程序，VS2010工程，演示了一对一的音视频通信功能，实现了AnyChat Windows SDK的基本开发流程；该版本提供了签名登录方式登录到AnyChat视频云平台的默认演示应用;
